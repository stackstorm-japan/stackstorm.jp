「StackStormを活用した運用自動化の実践」セミナー開催のお知らせ

2019/03/14 in using tags 勉強会

2019/03/19 飯田橋 IIJ-II にて、インターネットマルチフィード株式会社の杉本周氏による「StackStormを活用した運用自動化の実践」セミナー が開催されます。

講演者の杉本氏は、黎明期から StackStorm を活用した運用効率化に取り組んでおり、st2-docker など活用した運用 など新しいトレンドを活用した運用に長けた、国内有数の StackStorm のエキスパートとして知られています。

今回のセミナーでは、日本最大級の商用 IX サービス (JPNAP) の運用において、比較的最近リリースされた StackStorm の新機能 “Inquiries” (従来予め規定された通りの処理を行うワークフローの処理を、状況に応じて動的に処理を変更させられる機能) などを活用した運用自動化の実例をご紹介いただきます。

実施詳細

  • 日時：2019年3月19日18:00 (17:45 開場予定、19:30頃終了予定)
  • 場所：東京都千代田区富士見2-10-2 (株式会社インターネットイニシアティブ 13階 Opera2)
  • 参加費：無料
  • 詳細・参加申し込み：https://iijlab-seminars.connpass.com/event/124111/
